# Images Folder

## Logo
Place your `logo.png` file in this folder. The logo should be:
- Transparent background (PNG format)
- Recommended size: 200x200px or larger
- Should incorporate the "PODPEROS" branding with lips/pouch theme
- Red/Pink color scheme to match the website
