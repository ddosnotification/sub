## ğŸ”§ Setup & Deployment

### Local Development
1. Clone or download the repository
2. Open `index.html` in a modern web browser
3. No build process required - it's vanilla HTML/CSS/JS!

1. Create a PNG logo (500x500px recommended)
2. Save as `images/logo.png`

- **Colors**: Edit CSS variables in `:root` section of `style.css`

- **Brands**: Update brand names in the brands grid
